## Run Locally

Clone the project

```bash
  git clone https://github.com/Komik55/react-alif-task-app.git
```

Go to the project directory

```bash
  cd react-alif-task-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
